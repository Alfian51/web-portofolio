Kasir Caffe

Sistem Point of Sale (POS) Ringan & Modern untuk Operasional Kafe

Screenshot Tampilan Website

Overview Proyek

Kasir Caffe adalah sebuah aplikasi Point of Sale (POS) berbasis web terintegrasi yang dioptimalkan khusus untuk mendukung operasional harian kafe berskala mikro hingga kecil. Aplikasi ini mengutamakan performa pemrosesan pesanan yang instan, efisiensi manajemen data transaksi lokal, serta kemudahan pencetakan bukti bayar fisik.

Tujuan Utama

Menyediakan solusi pencatatan transaksi kasir digital secara mandiri, gratis, dan efisien tanpa memerlukan spesifikasi hardware komputer yang tinggi. Cukup dengan browser tablet atau laptop, pelaku usaha dapat langsung mengelola operasional penjualan harian mereka secara profesional.

Fitur Utama

  • Mesin Kasir POS Dinamis: Tambah, kurangi, hapus item pesanan langsung dari keranjang kasir secara real-time dengan pembaruan otomatis total biaya.
  • Kalkulator Kembalian & Pajak: Perhitungan kembalian tunai secara presisi untuk menghindari kesalahan input kasir saat bertransaksi.
  • Template Cetak Struk Mini: Desain cetak struk khusus yang disesuaikan secara otomatis (@media print) agar sesuai jika dicetak menggunakan printer thermal mini.
  • Manajemen Data Menu: Pengelompokan jenis menu (Makanan, Minuman, Camilan) yang tertata rapi untuk memudahkan pencarian menu dan pesanan.

Teknologi yang Digunakan

HTML5 CSS3 JavaScript Bootstrap 5 Firebase Hosting

Tantangan Pengembangan

Mengelola status data keranjang belanja (cart state management) agar berjalan lancar tanpa mengalami bug kalkulasi pembulatan nilai desimal JavaScript, serta merancang visualisasi cetak struk agar pas di berbagai lebar kertas printer kasir.

Pembelajaran Utama

Saya mendalami implementasi struktur data Array & Object bersarang (nested) di JavaScript, menguasai teknik dynamic print stylesheet CSS, serta memahami siklus deployment aplikasi web statis menggunakan Google Firebase Suite (Firebase Hosting).
